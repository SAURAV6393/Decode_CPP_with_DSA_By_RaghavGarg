## 📝 Assignment Tips

1. **Start Simple**: Understand single loop first
2. **Visualize**: Draw output before coding
3. **Test Cases**: Try different inputs
4. **Edge Cases**: Handle 0, negative, large numbers
5. **Optimize**: Reduce nested loops if possible
